## Kzap-page

Protótipo de landing page para uma plataforma de atendimento pelo whatsapp.
O objetivo é apresentar a plataforma e seus recursos para potenciais clientes.

## Tecnologias

* Html
* Css
* Javascript
* Bootstrap v4.3.1
* jQuery v2.1.0
* Font Awesome 4.7.0
* jQuery FlexSlider v2.7.1
* Owl Carousel v2.3.4
